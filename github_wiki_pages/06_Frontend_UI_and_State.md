# Frontend UI and State

## Product Views

### View 1: Skill Radar

- Before/after mastery overlay.
- Rapid perception of role fit and gap closure.
- Built for visual impact and comprehension speed.

### View 2: Three-Tab Pareto Interface

- Speed (lambda 2.0)
- Balance (lambda 1.0)
- Depth (lambda 0.2)

Each tab surfaces:

- Total hours.
- Coverage percentage.
- CVS.
- Interactive module timeline.

### View 3: Reasoning Trace Panel

- Technical mode: structured trace with verification evidence.
- HR mode: plain-language rationale.
- Provenance graph visualization.

### View 4: HR Dashboard

- NEXUS vs legacy CVS comparison.
- Compliance status.
- Standard vs adaptive path comparison.
- Catalog gap heatmap.
- Confidence interval signals.

## State Principles

- Streaming-first updates from SSE.
- Session-scoped generation state.
- Clear reset paths on new uploads.
- Deterministic rendering from backend payloads.

## UX Priorities from PRD

- Fast first meaningful result.
- Explainability for mixed audiences.
- Strong visual hierarchy for judges.
