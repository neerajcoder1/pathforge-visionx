# Evaluation Metrics and Demo Strategy

## Metric Framework

### Technical Sophistication

- Extraction quality and consistency.
- Engine correctness and responsiveness.

### Grounding and Reliability

- Hallucination prevention evidence.
- Verification pass behavior.

### Reasoning Trace

- Coverage of explainability artifacts.
- Clarity across technical and HR audiences.

### Product Impact

- CVS uplift vs legacy baseline.
- Hours saved and relevance improvements.

### UX Quality

- Path generation latency.
- Interaction quality of key views.

### Cross-Domain Scalability

- Distinct role families with same engine.

## Demo Storyline Emphasis

1. Show static onboarding pain.
2. Run adaptive generation.
3. Surface three-path tradeoff.
4. Show trace with technical and HR toggles.
5. Trigger diagnostic update and live adaptation.
6. Present business impact and compliance confidence.

## Demo Hardening from PRD

- Pre-cached profiles.
- Fallback model path.
- Backup recording.
- Offline-capable execution.
